# Rental Motor - Next.js + Firebase (Template)

Versi template sederhana yang siap Anda deploy ke **Vercel** dan simpan di **GitHub**.

## Cara pakai (singkat)

1. Install dependencies:
```bash
npm install
```

2. Tambahkan environment variables di `.env.local` (lihat `.env.example`).

3. Jalankan development:
```bash
npm run dev
```

4. Deploy:
- Push ke GitHub, lalu hubungkan repo ke Vercel.
- Tambahkan environment variables di dashboard Vercel.

## Struktur
- `app/` — pages & layout (Next.js App Router)
- `components/` — komponen UI
- `firebase.js` — konfigurasi Firebase
- `public/` — aset
